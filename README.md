# ThanhHoa Starter

ThanhHoa Starter is a template that helps to quickly initialize projects with ThanhHoaJS framework on Bun

## Installation

```bash
$ git clone https://github.com/thanhhoajs/starter.git project
$ cd project
$ bun install
$ bun dev
```
